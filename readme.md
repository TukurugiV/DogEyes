# DogEyes(犬の色覚再現プログラム)
## about me
- 犬はが見ている世界は、人間が見ている世界とはかなり異なる。
- その違いを以下に示す。
	- __色覚：人間は赤、緑、青の3色と、その中間色を認識できるが、犬は青と黄、その中間色しか認識できない。それ以外の色はグレーに見えると言われている(特に赤色がグレーになって見えているとされている)__
	- 視力：大抵の犬は近視(狩猟犬等は遠視である可能性もあるとされている)であると言われている。
	- 視野角：人間は左右の目を合わせて180度程度であるが、犬は250度程度とされている。
	- 犬しかない特徴
		- タペタム層：犬はわずかな光を視神経に伝える細胞を持っており、暗闇でも物を人試kすることができる。
		- 紫外線：最近の研究で、犬が紫外線を認識することができるのではないかということが言われ始めました
- 今回は色覚を再現することにした。
- 色覚といっても色だけではなく、露光度やコントラスト,明度等のさまざまな色に関する要素に要素に対して処理が行われてい(人間よりも若干暗く見える等)
